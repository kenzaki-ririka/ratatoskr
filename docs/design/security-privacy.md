# 隐私与合规设计

## 原则
- 最小化：仅在用户点击悬浮球时采集并发送必要上下文。
- 可控：不做自动发送；用户始终掌控是否使用候选。
- 透明：提供清晰的隐私说明与数据使用声明。

## 数据处理
- 采集：界面文本（不含图像/录屏）；不保存原始消息到持久化存储。
- 传输：HTTPS；请求体仅包含必要上下文与风格参数。
- 日志：仅匿名指标，禁止记录原始文本；可记录长度、耗时与点击率。

## 用户选择
- 设置项：可关闭悬浮球与 AI；可选择仅本地生成模式；可设置候选数量与风格偏好。
- 权限引导：说明用途与风险，支持随时撤销。

## 脱敏建议
- 对手机号、邮箱、地址等敏感模式进行掩码处理。
- 对身份信息进行替换标记（如“<姓名>”）。

## 合规与审计
- 明确用途：仅用于生成回复建议，辅助沟通。
- 数据保留：不保留原始上下文；指标数据设定保留窗口与清理策略。
- 变更管理：隐私政策更新需提示用户。
